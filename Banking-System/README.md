# Banking Management System

A backend banking system built using Oracle SQL and PL/SQL.

## Features
- Account creation and management
- Deposit and withdrawal operations
- Secure money transfer between accounts
- Transaction logging
- Data validation using triggers
- Modular design using PL/SQL packages

## Concepts Used
- Tables & Constraints
- Sequences
- Stored Procedures & Functions
- Triggers
- Transaction Control (COMMIT / ROLLBACK)
- Packages
